# Butik Portfolio

Портфолио frontend-разработчика **Butik** (`@butik_43`).

## Стек

- HTML / CSS / JavaScript
- Node.js API для отзывов (регистрация + общий список)
- Вложенные кейсы в `works/`

## Локальный запуск

Можно просто открыть `index.html` в браузере **или**:

```bash
node serve.js
# http://localhost:8765
```

### Отзывы (без своего сервера)

Раздел **Отзывы** работает из обычного HTML:

1. Регистрация / вход на сайте
2. Написать отзыв и оценку ★
3. Отзыв видят **все** посетители (общее облачное хранилище JSONBlob)
4. Один аккаунт = один отзыв (можно обновить или удалить)

Пароли хешируются в браузере (PBKDF2). Если облако недоступно — запасной режим: localStorage (только этот браузер).

## Структура

```text
index.html      # главная (SPA)
blog.html       # блог
js/reviews.js   # отзывы + auth (client-side)
serve.js        # опциональный локальный сервер
css/ js/ works/
```

## Деплой

Подойдёт **GitHub Pages**, Vercel, Netlify или любой статический хостинг — отзывы работают без Node backend.
