# Telegram-бот «Элегия» — за 5 минут

## Что делает бот

1. **Запись прямо в Telegram** (кнопки): услуга → мастер → дата → время → имя/телефон
2. **Мои записи** — список визитов и отмена кнопкой
3. Каталог услуг и мастеров, контакты
4. Принимает записи с сайта (`POST /api/bookings`)
5. Уведомляет **администратора** о новых записях
6. Админ: «🗂 Сегодня», «📑 Все записи»

### Меню бота

| Кнопка | Действие |
|--------|----------|
| 📅 Записаться | Мастер записи |
| 📋 Мои записи | Визиты + отмена |
| 💅 Услуги | Прайс |
| 👩‍🎨 Мастера | Команда |
| 📞 Контакты | Адрес и телефон |
| 🗂 Сегодня / 📑 Все записи | Только админ |

Команды: `/start` `/book` `/my` `/services` `/help` `/today` `/list`

## Шаг 1. Создать бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Имя: `Элегия Beauty Studio`
4. Username: например `elegiya_beauty_bot` (должен быть свободен, заканчивается на `bot`)
5. Скопируйте **токен** вида `7123456789:AAHxxxxxxxxxxxxxxxx`

## Шаг 2. Настроить `.env`

```bat
cd E:\elegiya-beauty\bot
copy .env.example .env
notepad .env
```

Пропишите:

```env
BOT_TOKEN=ваш_токен_от_BotFather
BOT_USERNAME=elegiya_beauty_bot
PORT=3001
ADMIN_CHAT_ID=
```

`BOT_USERNAME` — username **без** `@`, тот же что у BotFather.

## Шаг 3. Запустить

Дважды кликните **`start.bat`**  
или в терминале:

```bat
cd E:\elegiya-beauty\bot
npm.cmd install
npm.cmd start
```

Должно появиться:

```
🌿 Элегия API: http://localhost:3001
🤖 Бот @elegiya_beauty_bot запущен
```

## Шаг 4. Узнать ADMIN_CHAT_ID

1. Найдите своего бота в Telegram
2. Нажмите **Start** / отправьте `/start`
3. Отправьте `/id`
4. Скопируйте число (например `123456789`)
5. Вставьте в `.env`:

```env
ADMIN_CHAT_ID=123456789
```

6. Перезапустите `start.bat`

Теперь **все новые записи с сайта** будут приходить вам в Telegram.

## Шаг 5. Открыть сайт

1. Оставьте окно бота запущенным
2. Откройте `E:\elegiya-beauty\index.html` в браузере
3. В блоке записи справа статус должен стать: **Telegram-бот онлайн**
4. Создайте тестовую запись → сообщение придёт админу

## Клиент: получать подтверждения лично

Клиент в боте:

```
/start
/link +7 999 123-45-67
```

Телефон должен совпадать с тем, что указан при записи на сайте.  
После этого подтверждения идут и клиенту, и админу.

Либо кнопка на сайте **«Привязать телефон в боте»** открывает deep-link.

## Команды

| Команда | Кто | Описание |
|---------|-----|----------|
| `/start` | все | приветствие |
| `/link +7999…` | все | привязать телефон |
| `/my` | все | мои записи |
| `/cancel ID` | владелец/админ | отмена |
| `/id` | все | Chat ID |
| `/today` | админ | записи на сегодня |
| `/list` | админ | последние 10 |

## Если «Бот офлайн»

- Запущен ли `start.bat`?
- Открывается ли http://localhost:3001/api/health ?
- Не блокирует ли файрвол порт 3001?
- Сайт на `file://` иногда режет fetch — откройте через сервер:

```bat
cd E:\elegiya-beauty
python -m http.server 8080
```

и зайдите на http://localhost:8080

## API

| Метод | Путь | Назначение |
|-------|------|------------|
| GET | `/api/health` | статус |
| GET | `/api/config` | username, botUrl |
| POST | `/api/bookings` | новая запись |
| GET | `/api/bookings?phone=` | записи клиента |
| POST | `/api/bookings/:id/cancel` | отмена |

Данные: `bot/data/bookings.json`, `bot/data/users.json`
